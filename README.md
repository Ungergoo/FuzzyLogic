# Лазерный сканер с нечеткой логикой

Код для микроконтроллера ESP32, управляющий двухкоординатным сканирующим устройством с инфракрасным датчиком температуры MLX90614.

## Состав проекта

- `sketch_apr07a.ino` - прошивка для ESP32 (управление шаговыми двигателями, опрос датчика, нечеткая логика)

## Функции кода для ESP32

- Управление двумя шаговыми двигателями через драйверы ULN2003
- Чтение температуры с инфракрасного датчика MLX90614
- Обработка команд с ПК через последовательный порт
- Адаптивное управление временем измерения на основе нечеткой логики

## Как загрузить код в ESP32

1. Установить драйвер для преобразователя USB-UART (CP210x или CH340 в зависимости от платы)
2. Скачать и установить Arduino IDE с официального сайта arduino.cc
3. В Arduino IDE открыть Файл - Настройки и в поле "Дополнительные ссылки для менеджера плат" вставить:
   https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json
4. Открыть Инструменты - Плата - Менеджер плат, найти "esp32" и установить
5. Выбрать плату: Инструменты - Плата - ESP32 Arduino - ESP32 Dev Module
6. Установить необходимые библиотеки через Инструменты - Управлять библиотеками:
   - Adafruit MLX90614 Library
   - Adafruit BusIO
7. Подключить ESP32 к компьютеру через USB
8. В Инструменты - Порт выбрать COM порт, соответствующий подключенной плате
9. Открыть файл sketch_apr07a.ino в Arduino IDE
10. Нажать кнопку "Загрузка" (стрелка вправо)

## Подключение оборудования

Измените подключение к пинам платы либо в коде либо на самой плате.
